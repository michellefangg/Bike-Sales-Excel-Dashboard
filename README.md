### Dataset

This dashboard uses a fictional customer survey dataset containing demographic details, income, commute distance, education levels, and bike purchasing status.  
The data was cleaned and structured for analysis using Excel pivot tables.

---

### Dashboard Interactivity

- **Marital Status filter** ‚Üí compare purchases between married and single customers  
- **Region filter** ‚Üí segment buyers across Europe, North America, and Pacific  
- **Education Level filter** ‚Üí analyze purchase patterns by highest education completed  
- **Gender selector** ‚Üí switch income and purchase visuals by male/female  
- **Age Bracket filter** ‚Üí view purchase counts across age groups  
- **Commute Distance filter** ‚Üí track purchase behavior based on travel distance  
- **Purchased Bike toggle (Yes/No)** ‚Üí isolate buyers vs non-buyers across all charts  

---

### Features & Visuals

- Average income per purchase by gender  
- Purchase count by age bracket  
- Commute distance vs bike purchases  
- Excel dashboard built with PivotCharts + slicers for real-time filtering
